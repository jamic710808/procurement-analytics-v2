# V4 使用說明書

## 專案簡介
採購智能分析平台 V4.0 是一款整合型的互動式採購數據儀表板，支援 Excel 資料的上傳、下載與編輯，實現真正的資料驅動決策。

## 快速開始

### 啟動方式
直接在瀏覽器中開啟 `Procurement_Dashboard_V4.html` 檔案。

**支援瀏覽器**：
- Google Chrome 90+
- Microsoft Edge 90+
- Firefox 88+
- Safari 14+

### 主要功能

#### 1. 資料上傳
1. 點擊右上角「匯入 Excel」按鈕
2. 或前往「資料管理中心」點擊上傳區塊
3. 支援格式：`.xlsx`, `.xls`, `.csv`
4. 上傳成功後，所有頁面圖表將自動更新

#### 2. 資料下載
**下載範本**：
1. 前往「資料管理中心」
2. 點擊「下載資料範本」
3. 取得空白 Excel，依照格式填寫
4. 填寫完成後重新上傳

**匯出當前資料**：
1. 前往「資料管理中心」
2. 點擊「匯出目前資料」
3. 系統將下載包含所有已載入資料的 Excel

#### 3. 資料編輯器
1. 前往「資料管理中心」
2. 點擊「開啟資料編輯器」
3. 選擇工作表（如「支出明細」）
4. 直接在表格中修改數值
5. 點擊「儲存變更」

|------|----------|
| 決策總覽 | 核心 KPI 指標板、警示通知、月度趨勢 |

| 計畫偏差 | 超採/漏採追蹤、計劃 vs 實際對比 |
| 風險管控 | 地緣風險、高風險事件、緩解措施 |
| ESG 永續 | E/S/G 三維度評分、碳排放追蹤 |
| 合約管理 | 合約到期預警，有效期限追蹤 |

| 資料管理中心 | 上傳/下載/編輯的統一入口 |

### Excel 資料格式

請確保上傳的 Excel 包含以下工作表：

```
支出明細（必需）
├── 日期 (YYYY-MM-DD)
├── 供應商
├── 品類
├── 部門
├── 金額
├── 單價（可選）
└── 數量（可選）

供應商主檔
├── 供應商
├── 品類
├── 支出
├── 準時率 (0-100)
├── 合格率 (0-100)
├── ESG (A/B/C/D)
├── 風險 (低/中/高)
├── 評級 (A/B/C/D)
└── 評分 (0-100)

成本節約
├── 類型
├── 金額
├── 日期
└── 來源

品質數據
├── 日期
├── 供應商
├── 品項
├── 缺陷類型
├── 數量
├── 處理方式
└── 狀態

計畫與實際
├── 品類
├── 計劃量
├── 實際量
└── 去年同期

ESG評分
├── 供應商
├── E (環境分數)
├── S (社會分數)
├── G (治理分數)
└── 總分

合約管理
├── 合約編號
├── 供應商
├── 品類
├── 簽約日
├── 到期日
├── 金額
└── 狀態
```

### 疑難排解

| 問題 | 解決方案 |
|------|----------|
| 上傳失敗 | 確認 Excel 格式正確，工作表名稱需完全匹配 |
| 圖表空白 | 檢查上傳的資料是否包含必要欄位 |
| 檔案過大 | 建議單一 Excel 不超過 10MB |
| 中文亂碼 | 確認檔案編碼為 UTF-8 |

### 快捷鍵

| 快捷鍵 | 功能 |
|--------|------|
| `Ctrl + U` | 開啟上傳對話框 |
| `Ctrl + S` | 匯出當前資料 |
| `Esc` | 關閉彈窗 |
