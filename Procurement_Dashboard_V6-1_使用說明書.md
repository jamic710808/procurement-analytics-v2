## 2. 快速開始

### 2.1 開啟方式

```bash
# 直接以檔案總管開啟（最簡單）
雙擊 "Procurement_Dashboard_V6-1.html"

# 或使用瀏覽器開啟
Chrome -> 右上角選單 -> 開啟檔案 -> 選擇 Procurement_Dashboard_V6-1.html
```

### 2.2 首次使用

1. 開啟檔案後，系統會顯示「決策總覽」頁面
2. 若有範例資料，按鈕會顯示「範例資料」
3. 若要使用自己的資料，點擊「匯入 Excel」按鈕

## 4. 主要功能

### 4.1 資料匯入（Excel）

**支援格式**：`.xlsx`、`.xls`、`.csv`

**匯入步驟**：
1. 點擊右上角「匯入 Excel」
2. 選擇您的 Excel 檔案
3. 系統會顯示匯入進度
4. 匯入成功後，指標會自動更新

**Excel 格式要求**：
- 工作表名稱需與系統匹配（請參考系統內的資料管理中心）
- 第一列為標題列
- 日期格式建議為 YYYY-MM-DD
- 數值欄位避免包含單位符號（如 $、%）

### 4.2 主題切換

系統支援亮色/深色雙主題：

| 按鈕位置 | 操作方式 |
|----------|----------|
| 右上角（月亮圖示）| 點擊即可切換 |

**提示**：主題設定會影響 PDF 匯出的外觀。

### 4.3 企業級 PDF 匯出（V6.1 新功能）

這是 V6 版本的旗艦功能，支援多頁批量匯出。

#### 4.3.1 開啟匯出設定

1. 點擊右上角「進階匯出」按鈕
2. 彈出「企業級匯出設定 (V6.0)」Modal

#### 4.3.2 選擇匯出頁面

在「匯出範圍」區塊：
- **全選**：點擊後勾選所有頁面
- **清除**：取消所有頁面
- **個別選擇**：直接勾選要匯出的頁面

#### 4.3.3 自訂浮水印

在「自訂浮水印」輸入框輸入文字（選填）：
- 例如：`機密文件`
- 例如：`僅供內部參考`

#### 4.3.4 版面與格式選項

| 選項 | 說明 | 預設值 |
|------|------|--------|
| 匯出主題跟隨目前畫面 | 自動偵測當前主題 | 啟用（disabled） |
| 加入匯出截期與戳記 | 在 PDF 加入匯出日期時間 | 啟用 |
| 直式版面 | A4 直向（預設為橫式） | 停用（橫式） |
| 啟用高品質渲染 | 2x 像素密度（解析度更高） | 啟用 |

#### 4.3.5 執行匯出

1. 確認所有設定正確
2. 點擊「開始匯出」按鈕
3. 系統會顯示「正在產生高品質 PDF，請稍候...」
4. 視頁面數量與網路速度，可能需要 10-60 秒
5. 完成後瀏覽器會自動下載 PDF 檔案

#### 4.3.6 匯出成功標準

- PDF 頁面數量等於選擇的頁面數量
- 主題顏色與當前畫面一致（亮色→淺米色背景，深色→深藍色背景）
- 所有圖表清晰可見
- 中文文字正確顯示

### 4.4 圖表互動

| 互動方式 | 說明 |
|----------|------|
| 懸停 | 滑鼠移至圖表元素顯示數值 |
| 點擊 | 某些圖表支援點擊鑽取詳細資料 |
| 拖曳 | 雷達圖支援拖曳旋轉視角 |

## 6. 疑難排解

### 6.1 PDF 匯出主題顏色錯誤

**問題**：匯出 PDF 的主題與當前畫面不符

**解決方式**：
1. 確認當前主題是亮色或深色
2. 關閉匯出 Modal
3. 重新點擊「進階匯出」
4. 再次嘗試匯出

### 6.2 圖表無法顯示

**檢查項目**：
1. 是否已匯入資料
2. 網路連線是否正常（需連線 Chart.js CDN）
3. 瀏覽器是否封鎖了 JavaScript

**解決方式**：
1. 按 `F12` 開啟開發者工具
2. 切換至「Console」標籤
3. 查看是否有錯誤訊息

### 6.3 Excel 匯入失敗

**常見原因**：
- 檔案格式不正確（僅支援 .xlsx/.xls/.csv）
- 工作表名稱與系統不符
- 欄位標題與預期不符

**解決方式**：
1. 確認 Excel 格式符合範本
2. 嘗試使用系統提供的範本資料
3. 聯繫技術支援

### 6.4 匯出時當機

**原因**：記憶體不足（大量頁面 + 高品質模式）

**解決方式**：
1. 減少選擇的頁面數量
2. 取消「啟用高品質渲染」
3. 分批匯出（每次 3-5 頁）

## 7. 資料格式說明

### 7.1 必要的資料工作表

| 工作表名稱 | 必要性 | 說明 |
|-----------|--------|------|
| spend | 必要 | 支出明細資料 |
| supplier | 建議 | 供應商主檔 |
| quality | 建議 | 品質檢驗資料 |
| delivery | 建議 | 交貨記錄 |
| contract | 選填 | 合約資料 |

### 7.2 欄位命名範例

```
spend 工作表：
- 日期 (date)
- 供應商 (supplier)
- 品類 (category)
- 金額 (amount)
- 幣別 (currency)

supplier 工作表：
- 供應商名稱 (name)
- 等級 (tier)
- 國家 (country)
- ESG 評分 (esg_score)
```
