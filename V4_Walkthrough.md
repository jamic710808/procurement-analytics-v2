# V4 Walkthrough

## 第一次使用引導

### Step 1：開啟儀表板
1. 雙擊 `Procurement_Dashboard_V4.html`
2. 等待載入（約 3-5 秒）
3. 預設顯示「決策總覽」頁面

### Step 2：認識介面
- **左側欄**：導航選單，12 個分析頁面
- **頂部工具列**：篩選器、資料狀態、功能按鈕
- **主內容區**：圖表、KPI 卡片、資料表格

### Step 3：嘗試上傳資料
1. 點擊右上角「匯入 Excel」
2. 選擇您已準備好的 Excel 檔案
3. 等待上傳完成
4. 觀察頁面自動更新

### Step 4：下載範本
1. 前往「資料管理中心」（最底部導航）
2. 點擊「下載資料範本」
3. 取得空白範本
4. 依照格式填寫您的資料
5. 重新上傳

### Step 5：編輯資料
1. 在「資料管理中心」點擊「開啟資料編輯器」
2. 選擇要編輯的工作表
3. 直接修改儲存格內容
4. 點擊「儲存變更」
5. 返回各頁面查看更新結果

### Step 6：使用篩選器
1. 在頂部工具列選擇年份（2024/2025）
2. 選擇季度（Q1-Q4）
3. 選擇品類（電子零組件等）
4. 所有頁面將連動更新

### Step 7：情境模擬
1. 前往「情境模擬」頁面
2. 拖動價格/供應商/需求滑桿
3. 觀察預測結果即時變化
4. 閱讀系統提供的策略建議

## 快速參考

### 頁面跳轉
- 點擊左側導航即可切換頁面
- 或使用頂部「資料管理」按鈕直達資料中心

### 圖表互動
- 滑鼠懸停顯示詳細數值
- 點擊圖例可顯示/隱藏數據系列
- 支援縮放和平移（部分圖表）

### 資料操作
| 操作 | 位置 | 說明 |
|------|------|------|
| 上傳 | 右上角 / 資料管理中心 | 支援 .xlsx/.xls/.csv |
| 下載範本 | 資料管理中心 | 取得空白填寫格式 |
| 匯出資料 | 資料管理中心 | 下載當前所有資料 |
| 編輯 | 資料管理中心 | 網頁直接修改 |
