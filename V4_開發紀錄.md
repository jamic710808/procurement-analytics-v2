# V4 開發紀錄 (Development Record)

## 專案標題：採購智能分析平台 V4.0

### 開發目標
整合 V1、V2、V3 版本的精華，並新增「Excel 上傳下載互動功能」，實現真正的資料驅動儀表板。

### 版本演進

| 版本 | 發布時間 | 核心特色 |
|------|----------|----------|
| V1 | 2024-Q3 | 基礎 8 頁面，深色玻璃擬態 |
| V2 | 2024-Q4 | 新導航架構，年度/品類篩選器，Plotly 圖表 |
| V3 | 2025-Q2 | Vizro 現代化框架，Plotly 圖表 |
| **V4** | **2025-Q4** | **Chart.js + SheetJS，Excel 上下傳，12 頁整合** |

### 採用技術棧
- **視覺化**：Chart.js 4.4.0（全域 `window.Chart`）
- **Excel 處理**：SheetJS (cdn.sheetjs.com/xlsx-0.20.3)
- **UI 設計**：深色玻璃擬態 (#0f172a 底色)
- **圖標**：FontAwesome 6.5 (CDN)
- **部署**：單一 HTML 檔案，無需 build

### V4 開發歷程

#### Phase 1：基礎架構 (2025-Q4)
- 統一 V1/V2/V3 的設計語言
- 建立全局狀態管理 (`AppData`)
- 實作 12 頁面架構

#### Phase 2：核心功能
- **上傳功能**：使用 SheetJS 解析 .xlsx/.xls/.csv
- **下載功能**：匯出當前資料為 Excel
- **範本下載**：提供空白範本供使用者填寫
- **資料編輯器**：網頁內直接檢視/編輯資料

#### Phase 3：頁面實作
1. 決策總覽 — KPI 儀表板 + 警示通知
2. 支出分析 — Spend Cube 三維度
3. 供應商矩陣 — 四象限 + 評分排名
4. 成本節約 — PPV / 價格差異
5. 品質交付 — SPC 管制圖
6. 計畫偏差 — 超採/漏採追蹤
7. 風險管控 — 地緣風險 + 事件
8. ESG 永續 — E/S/G 評分
9. 合約管理 — 到期預警
10. 採購成熟度 — 五階段模型
11. 情境模擬 — What-If 分析
12. 資料管理中心 — 上下傳樞紐

### 重大技術決策

1. **Chart.js 而非 Plotly**：避免 Vizro 的複雜依賴，保持輕量
2. **SheetJS CDN**：使用 cdn.sheetjs.com 而非 cdnjs（避免 404）
3. **全域 Chart 物件**：Chart.js UMD 全局為 `window.Chart`
4. **單一 HTML**：所有 CSS/JS 內嵌，簡化部署

### 已知限制
- 情境模擬器為前端計算，實際使用需結合後端
- 資料編輯器僅支援基本文字編輯
- 尚不支援 PDF 匯出（V5 規劃）
