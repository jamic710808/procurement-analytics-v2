# 現代化智能採購戰情室：使用說明書

## 專案簡介
本儀表板 (Dashboard) 提供全新的採購分析體驗，將原本散落的 Excel 檔案與 Word 報告，轉化為具備「深色玻璃擬態 (Dark Glassmorphism)」風格的視覺化互動戰情室。旨在協助高階主管與採購人員即時掌握支出、風險、品質與營運效率。

## 檔案結構
- `index.html`：儀表板入口與首頁。
- `css/style.css`：核心視覺樣式與版面配置。
- `js/data_engine.js`：資料狀態管理與檔案解析引擎。
- `js/dashboard.js`：Chart.js 圖表渲染與前端互動邏輯。
- `generate_sample_data.py`：範例數據產生器 (Python 腳本)。

## 如何啟動與使用
1. **開啟儀表板**
   直接在瀏覽器 (例如 Google Chrome 或 Edge) 中雙擊打開 `index.html`。
   
2. **檢視各分析面板**
   點擊左側側邊欄 (Sidebar) 的不同項目以切換頁面，包含：
   - 決策總覽 (Executive Summary)
   - 支出多維分析 (Spend Cube)
   - 供應商績效與風險矩陣
   - 成本節約與價格差異
   - 品質與交付追蹤
   - 計畫偏差管理
   - 營運與流程效率
   - 情境決策模擬器

3. **產生範例數據**
   如果您已安裝 Python 環境以及對應的套件 (`pandas`, `numpy`, `openpyxl`)，請在終端機中執行：
   ```bash
   python generate_sample_data.py
   ```
   系統將在同一目錄下產生一份 `Procurement_Sample_Data.xlsx`。

4. **動態數據匯入測試**
   點擊儀表板右上角的「匯入/更新資料」按鈕，選擇剛剛產生的 `Procurement_Sample_Data.xlsx`。
   系統會透過 `data_engine.js` 解析檔案，並即時更新畫面上（如 Executive Summary 頁面）的指標板與狀態，呈現互動連動效果。

## 後續維護建議
- 若欲置換為貴公司真實資料，可由開發人員擴充 `data_engine.js` 中的 sheet mapping 邏輯。
- 欲添加新圖表時，可使用 HTML 中的 `canvas` 標籤，並於 `dashboard.js` 中實例化 Chart.js。
